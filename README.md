#  Employee Payroll Management System

A complete **Java OOP + Frontend Project** for managing employees, salaries, and payroll operations. This project is designed for beginners and intermediate developers to understand **Object-Oriented Programming concepts in Java** with a simple frontend UI.

---

#  Project Overview

The **Employee Payroll Management System** helps organizations manage employee salary records efficiently.

Users can:

- Add Employees
- Add Managers with Bonus
- View All Employees
- Remove Employees
- Calculate Total Salary
- Use Frontend UI for quick employee management

---

#  Features

## Backend Features (Java)
```
- Add Normal Employee  
- Add Manager with Bonus  
- View Employee List  
- Remove Employee  
- Salary Calculation  
- Console Based Menu  
- OOP Concepts Implemented
```
## Frontend Features
```
- Add Employee via UI  
- Display Employee List  
- Remove Employee  
- Clean Responsive Design
```

#  OOP Concepts Used

## 1. Encapsulation
Private variables with getter methods.

## 2. Inheritance
`Manager` class extends `Employee`

## 3. Polymorphism
Overriding `calculateSalary()` method.

## 4. Abstraction
Payroll system hides internal logic.

## 5. Classes & Objects
Core project structure based on classes.
